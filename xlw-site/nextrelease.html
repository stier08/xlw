Download Page

The latest XLW release is the 0.2.1 version. It includes the following files:

  1. XLW
    • Win32
      • XLW-0.2.1-full-inst.exe (11.6 MB): this is the main Win32 distribution file. It includes binaries, source code, documentation, and examples.
      • XLW-0.2.1-src.zip (890 KB): zipped source code.
      • XLW-0.2.1-light-inst.exe (1.6 MB): small installer with binaries only.
      • XLW-0.2.1-debug.zip (4.5 MB): debug libraries.
    • Unix
      • XLW-0.2.1-src.tar.gz (597 KB): gzipped source code.
      • XLW-0.2.1-src.tar.bz2 (457 KB): source code zipped with bzip2.
      • http://ftp.us.debian.org/debian/pool/main/q/xlw: Debian package (the .us can be replaced by .jp, .uk, .de, .it, etc.)
