Download Page

The latest XLW release is the 1.1 version. It includes the following files:

  • Library
    • XLW-1.1.0-full-inst.exe (MB): this is the main Win32 distribution file. It includes binaries, source code, documentation, and examples.
    • XLW-1.1.0-src.zip (KB): zipped source code.
    • XLW-1.1.0-light-inst.exe (MB): small installer with binaries only.
  • Documentation
    • XLW-1.1.0-docs-html.zip (MB): zipped html documentation.
    • XLW-1.1.0-docs-refman.chm (MB): WinHelp documentation.
    • XLW-1.1.0-docs-refman.chm.zip (MB): zipped WinHelp documentation.
    • XLW-1.1.0-docs-refman.pdf (MB): pdf documentation.
    • XLW-1.1.0-docs-refman.pdf.zip (MB): zipped pdf documentation.
    • XLW-1.1.0-docs-refman.ps (MB): ps documentation.
    • XLW-1.1.0-docs-refman.ps.zip (KB): zipped ps documentation.
    • XLW-1.1.0-docs-refman.dvi (MB): dvi documentation.
    • XLW-1.1.0-docs-refman.dvi.zip (KB): zipped dvi documentation.
