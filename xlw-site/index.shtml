XLW - A C++ Wrapper for Excel

XLW implements a simple API which you can use to customize Excel with your own worksheet formulas and menu items.


The project includes the InterfaceGenerator utility which parses the header files of your C++ application and automatically generates the corresponding XLW Excel addin.


XLW supports new features introduced by Excel 2007 such as multithreading, larger spreadsheets, and Unicode strings.


XLW version 3 is currently in beta status. You're invited to download and test the software and report any problems in the bug tracker.
