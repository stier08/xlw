Documentation

Reference Manual

This manual documents all of the classes in xlw. The documentation is autogenerated from comments in the source code files using the Doxygen utility. All of the information has been completely updated and rewritten for version 3. The reference manual is available from the downloads page in chm format (Windows compiled help file), or may be browsed online at the link below:

http://xlw.sourceforge.net/refman/

Design Document for xlw Version 2

Includes a description of the InterfaceGenerator module which parses the header files of your application and autogenerates the source code for the corresponding xlw addin. All of the information remains relevant for version 3, and the document has been updated to reflect changes in path names etc. This document is available from the downloads page as a pdf file, or may be browsed online at the link below:

http://xlw.sourceforge.net/version2/index.html

Design Document for xlw Version 3

Includes an overview of the enhancements made in support of Excel 2007's new features. This document has been added to the reference manual mentioned above. Here is the link to the online version of the document:

http://xlw.sourceforge.net/refman/version3.html
