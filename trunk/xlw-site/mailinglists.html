Mailing Lists

Choose a list to browse, search, and post messages.
  • xlw-users:
    XLW users general discussion list. This list is for everything related to the XLW usage and development.
    (go to Archives, Subscribe/Unsubscribe/Preferences)
  • xlw-announce:
    XLW announcement list (very low traffic). You cannot post to this list. If you have an announcement for the XLW community, please post it on xlw-users: the XLW admin will forward it to xlw-announce if appropriate.
    (go to Archives, Subscribe/Unsubscribe/Preferences)
  • xlw-cvs:
    XLW CVS checkin messages (very high traffic). Every time the XLW CVS code changes an email documenting the change is automatically generated. Suggested for XLW developers only.
    (go to Archives, Subscribe/Unsubscribe/Preferences)
